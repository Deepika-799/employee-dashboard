# Employee Portal Management System

## Overview

The Employee Portal Management System is a web-based application designed to help employees efficiently manage their daily work activities, attendance, tasks, performance, and profile information. The platform provides a centralized dashboard where employees can monitor their productivity and stay updated with their work progress.

---

## Features

### 📊 Dashboard

Provides a comprehensive overview of employee activities and work statistics.

**Key Functions:**

* Employee activity summary
* Task overview
* Performance insights
* Productivity monitoring
* Quick access to portal features
